Infrastructure สำหรับโรงเรียน thatako - URL shortener, QR codes, subdomains และ public API สำหรับนักเรียนและครู

ย่อลิ้งก์ GitHub

บริการ

URL Shortener (go.thatako.net)

บริการย่อลิ้งก์สำหรับนักเรียนและครู สร้าง slug ที่จำง่ายและ QR code สำหรับแจกในห้องเรียน

กำลังพัฒนา - ใช้งานได้บางส่วนแล้วที่ thatako.net/short

GET https://go.thatako.net/{slug}
→ 302 redirect to target URL

Subdomains สำหรับนักเรียน

แผนในอนาคตจะมีการให้ subdomain แก่นักเรียนที่มีความสามารถแล้วอยากเขียน website ของตัวเองแต่ไม่มีโดเมนใช้ สามารถขอ subdomain ของเราได้ฟรี เช่น [name].id.thatako.net

ยังอยู่ในขั้นตอนการวางแผน ยังไม่เปิดให้สมัคร
