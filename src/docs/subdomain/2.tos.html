ภาพรวม

บริการ id.thatako.net ให้ subdomain ฟรีแก่นักเรียนและครูโรงเรียนท่าตะโกพิทยาคม ในรูปแบบ [name].id.thatako.net เพื่อใช้ในการเรียนรู้ พัฒนาโปรเจกต์ส่วนตัว และสร้างพอร์ตโฟลิโอ บริการนี้ดำเนินการโดยนักเรียนอาสาสมัคร (@aitji) ภายใต้โครงการ thatako.net

ข้อกำหนดนี้อาจเปลี่ยนแปลงได้โดยไม่แจ้งล่วงหน้า การใช้งานบริการต่อเนื่องหลังจากมีการเปลี่ยนแปลงถือว่ายอมรับข้อกำหนดฉบับใหม่

ผู้มีสิทธิ์ใช้งาน

บริการนี้เปิดไม่ได้เปิดเพียงแค่นักเรียนและครูโรงเรียนท่าตะโกพิทยาคม บุคคลากรภายนอกสามารถนำไปใช้ได้ด้วยเช่นกัน โดยต้องมีบัญชี GitHub สำหรับการยืนยันตัวตนและการจัดการ DNS record

การใช้งานที่อนุญาต

  • โฮสต์เว็บไซต์ส่วนตัว พอร์ตโฟลิโอ หรือโปรเจกต์การเรียนรู้
  • ทดสอบ web application หรือ static site บน production domain
  • ใช้ TXT record สำหรับ domain verification ของบริการต่าง ๆ
  • ใช้ MX record สำหรับรับ email ผ่านโดเมนนี้
  • ใช้งานจากที่ไหนก็ได้ ไม่จำกัดเฉพาะบริเวณโรงเรียน
  • เพิ่ม co-owner เพื่อให้ผู้อื่นช่วยจัดการ DNS record ร่วมกัน

การใช้งานที่ต้องห้าม

การกระทำต่อไปนี้ถือเป็นการละเมิดข้อกำหนดและอาจนำไปสู่การระงับ subdomain ทันที

  • เนื้อหาต้องห้าม
  • เนื้อหา NSFW สื่อลามก เนื้อหาสำหรับผู้ใหญ่ หรือเนื้อหาที่ไม่เหมาะสมทุกประเภท
  • เนื้อหาผิดกฎหมาย เนื้อหาที่ผิดกฎหมายไทย ละเมิดลิขสิทธิ์ หรือละเมิดสิทธิ์ส่วนบุคคลของผู้อื่น
  • การหลอกลวง Phishing, social engineering หรือเว็บไซต์ที่ออกแบบมาเพื่อหลอกลวง
  • Malware / Exploit แจกจ่าย malware, ransomware, virus หรือเนื้อหาที่ออกแบบมาเพื่อโจมตีระบบ
  • การใช้งานผิดวัตถุประสงค์
  • จอง namespace ลงทะเบียน subdomain จำนวนมากโดยไม่มีวัตถุประสงค์การใช้งานจริง
  • แก้ไข subdomain ผู้อื่น แก้ไขหรือลบ subdomain ที่ตัวเองไม่ใช่ owner หรือ co-owner
  • ปลอม subdomain ของโรงเรียน สร้างเว็บที่แสร้งว่าเป็นเว็บทางการของโรงเรียนท่าตะโกพิทยาคม
  • ใช้ในเชิงพาณิชย์ที่ไม่เหมาะสม โฆษณาหรือขายสินค้าผิดกฎหมาย
    *อนุญาติให้เปิดสร้างรายได้กับ ads network ต่าง ๆ ได้เช่น Google Adsense แต่ไม่สร้างบริการที่ใช้การสร้างโดเมนฟรีนี้เป็นการหารายได้

ความเป็นเจ้าของและ Co-owner

ผู้ที่ลงทะเบียน subdomain จะเป็น Primary Owner ซึ่งมีสิทธิ์เต็มในการแก้ไขและลบโดเมนนั้น สามารถเพิ่ม Co-owner ได้เพื่อให้ผู้อื่นช่วยจัดการ DNS record ร่วมกัน

ทั้ง owner และ co-owner สามารถแก้ไข DNS record ผ่านหน้าเว็บได้โดยตรง
หากมีการแก้ไขผ่าน GitHub โดยตรง ระบบจะตรวจสอบ ownership อัตโนมัติ
และ tag @aitji หากผู้แก้ไขไม่ใช่ owner หรือ co-owner

การระงับและลบโดเมน

maintainer (@aitji) มีสิทธิ์ระงับหรือลบ subdomain ในกรณีต่อไปนี้ โดยไม่จำเป็นต้องแจ้งล่วงหน้า

  • พบการละเมิดข้อกำหนดข้างต้น
  • ได้รับรายงาน abuse ที่มีหลักฐานชัดเจน
  • เว็บไซต์ปลายทางไม่ตอบสนองเป็นเวลานานและ owner ไม่ตอบสนองต่อการติดต่อ
  • จำเป็นต้องใช้ namespace นั้นสำหรับบริการของโรงเรียน

ข้อจำกัดความรับผิดชอบ

บริการนี้ให้บริการ "ตามที่เป็น" (as-is) โดยไม่มีการรับประกัน uptime หรือความพร้อมใช้งาน โครงการ thatako.net และ maintainer ไม่รับผิดชอบต่อความเสียหายใด ๆ
ที่เกิดจากการใช้งานบริการนี้ รวมถึงการสูญหายของข้อมูล หรือการหยุดให้บริการ

เนื้อหาบน subdomain ทั้งหมดเป็นความรับผิดชอบของ owner โดยตรง โครงการ thatako.net ไม่ได้เป็นผู้ตรวจสอบหรือรับรองเนื้อหาบน subdomain ใด ๆ
