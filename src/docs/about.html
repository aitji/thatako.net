เกี่ยวกับ thatako.net

aitji · อัปเดตล่าสุด 2026

thatako.net คือ infrastructure ด้านเว็บสำหรับโรงเรียนท่าตะโกพิทยาคม ออกแบบให้เรียบง่าย โปร่งใส และควบคุมได้เองทั้งหมด ตั้งแต่ DNS, edge routing, short URL ไปจนถึง public API สำหรับนักเรียนและครู

หน้าหลัก GitHub

thatako.net คืออะไร

โปรเจกต์นี้เริ่มจากแนวคิดง่าย ๆ ว่าโรงเรียนควรมีระบบเว็บที่ จัดการได้เอง ไม่ต้องพึ่ง third-party platform มากเกินไป และสามารถขยายต่อได้ในอนาคต ไม่ว่าจะเป็น subdomain สำหรับนักเรียน หรือบริการ API สำหรับงานกิจกรรมต่าง ๆ

เป้าหมายหลักคือสร้าง platform กลางที่นักเรียนสามารถทดลอง พัฒนา และใช้งานจริงได้ในสภาพแวดล้อม production

เว็บไซต์ภายใต้ระบบ

  • pr.thatako.net - งานประชาสัมพันธ์
  • council.thatako.net - สภานักเรียน
  • go.thatako.net - URL Shortener
  • [name].id.thatako.net - Subdomain สำหรับนักเรียนหรือคุณครู
